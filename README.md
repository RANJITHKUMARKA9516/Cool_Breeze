# ❄️ Cool Breeze – Multi-Agent AI Customer Support System

An AI-powered customer support application built with **Python**, **Django**, and **Anthropic Claude API**. The system uses a **Multi-Agent AI architecture** and **Retrieval-Augmented Generation (RAG)** to automate customer support, retrieve company knowledge, and generate intelligent, context-aware responses.

---

## 🚀 Features

* 🤖 Multi-Agent AI Architecture
* 💬 AI-Powered Customer Support
* 🧠 Anthropic Claude API Integration
* 📚 Retrieval-Augmented Generation (RAG)
* 🔍 ChromaDB Vector Database
* 📦 Customer Order Lookup
* 📄 Warranty & Refund Policy Retrieval
* 🏢 Company Knowledge Base Search
* ⚠️ Risk Assessment Agent
* 👨‍💼 Manager Approval Workflow
* 📊 AI Interaction Dashboard
* 🔐 Secure Environment Variable Configuration

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/Cool_Breeze.git
cd Cool_Breeze
```

### 2. Create a virtual environment

**Windows**

```bash
python -m venv env
env\Scripts\activate
```

**macOS / Linux**

```bash
python3 -m venv env
source env/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Configure environment variables

Create a `.env` file in the project root.

Example:

```env
ANTHROPIC_API_KEY=your_api_key
SECRET_KEY=your_secret_key
DEBUG=True
```

### 5. Apply migrations

```bash
python manage.py migrate
```

### 6. Run the development server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
